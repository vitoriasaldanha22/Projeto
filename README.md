# Projeto

## Resumo

Este projeto é uma API que implementa um CRUD básico utilizando Java e Spring Boot.

## Tecnologias Utilizadas

- Java
- Spring Boot

## Funcionalidades

- Criação, leitura, atualização e remoção de dados (CRUD)

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/vitoriasaldanha22/Projeto.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd Projeto
   ```
3. Execute a aplicação:
   ```bash
   ./mvnw spring-boot:run
   ```
   ou
   ```bash
   mvn spring-boot:run
   ```
